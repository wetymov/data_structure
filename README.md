## Бинарное дерево
----
### Реализация
Реализация сделана: `добавление в дерево`, `удаление элемента из дерева`, `центрированый обход дерева`, `удаление дерева из памяти`

### Тесты
Тесты написаны для: `центрированый обход дерева`

### Асимптотика
`Add` : $O(\log n)$ до $O(n)$
`Remove` : $O(\log n)$ до $O(n)$
`InOrder` : $O(n)$
